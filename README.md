Welcome to my first desktop app!

This is a straightforward music player that allows the user to upload their own MP3 files to be played, built using Electron.
Currently this is desktop only and is not made for mobile devices.

To Use:
1. Select v1.0.0 under Releases
2. Click my-electron-app-1.0.0.Setup.exe file to download
3. Your Windows Defender/Antivirus program may pop up with a window asking you to verify before you can run it
4. Once installed, you can listen to music by uploading your own MP3 files and pressing play

For future releases I'd like to have the option to build a dynamic visual playlist out of MP3 uploads, and include an audio progress bar.
Please let me know if you encounter any issues or have any questions.
